#  AI Powered Smart Resume Analyzer

A modern web application that analyzes PDF resumes using **Google Gemini AI** and provides instant professional feedback.

##  Features
- Overall Score, ATS Score & Job Match Percentage
- Skill extraction (Technical + Soft skills)
- Strengths and Areas to Improve
- Actionable suggestions
- Modern lite blue UI with Dark/Light mode
- Download analysis report

## Tech Stack
- **Frontend**: Streamlit
- **Backend**: Python
- **AI**: Google Gemini 2.5 Flash
- **PDF Parser**: PyMuPDF
